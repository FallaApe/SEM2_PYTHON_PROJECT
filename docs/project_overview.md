# Project Overview

This project is a simple GUI-based network scanner built using Python.

The tool allows users to scan a target IP address and specified port range
using Nmap and view the results through a graphical interface.

The goal of the project is to demonstrate:

- Python GUI development
- Network scanning concepts
- Integration with external security tools
- Modular Python project structure

The scanner works by taking user input from the GUI and passing it to the
Nmap scanning engine through the python-nmap library. The results are then
parsed and displayed in the interface.
